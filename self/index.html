HTML5 Web Storage Example

Local Storage

Local storage allows you to store data that persists even after closing the browser.

Session Storage

Session storage stores data for the duration of the session. Once the tab or browser is closed, the data is lost.

Web Storage Methods and Properties

  • setItem(key, value): Stores a key-value pair.
  • getItem(key): Retrieves the value associated with the key.
  • removeItem(key): Removes the key-value pair.
  • clear(): Clears all storage data.
  • length: Returns the number of key-value pairs.
  • key(index): Returns the key at the specified index.
